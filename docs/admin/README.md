# Документация администратора

Этот раздел предназначен для установки, настройки и обслуживания RD4.

## Данные приложения

По умолчанию используется SQLite:

```text
Windows: %APPDATA%\RD4\rd4.db
Linux/macOS: ~/.rd4/rd4.db
Development override: RD4_DATA_DIR=.rd4
```

В папке данных хранятся:

- база данных;
- логи;
- пользовательские рабочие файлы, если они создаются приложением.

## База данных

Текущий рабочий backend — SQLite.

Архитектурно проект может быть подготовлен к PostgreSQL, потому что доступ к данным идет через SQLAlchemy и слой репозиториев. Полный переход на PostgreSQL потребует отдельной настройки `DATABASE_URL`, драйвера, миграций и процедур резервного копирования.

MySQL возможен технически, но для этого проекта PostgreSQL предпочтительнее из-за более предсказуемой работы транзакций, типов данных и миграций.
