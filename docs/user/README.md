# Пользовательская документация

Этот раздел описывает ежедневную работу оператора в RD4.

## Основные разделы приложения

- `Договоры` — реестр договоров, поиск, фильтры, создание, редактирование, печать.
- `Платежи` — оплаты, возвраты и распроведение платежей в карточке договора.
- `Акты` — акты выполненных услуг, печать актов и талонов.
- `Справочник услуг` — дерево папок и медицинских услуг.

## Вход в систему

Окно входа содержит:

- выбор пользователя;
- поле пароля;
- кнопку `Войти`;
- служебный блок с папкой данных и путем к базе.

При первом запуске пустой базы создается пользователь:

```text
login: admin
password: admin
```

## Договоры

На главной странице доступны:

- создание договора;
- создание договора на основе выбранного;
- открытие карточки договора;
- редактирование;
- печать договора из контекстного меню;
- soft-delete договора;
- фильтры по периоду, типу оплаты, балансу и видимости.

Удаленные договоры скрываются из активного списка, но доступны через фильтр видимости.

## Платежи

В карточке договора вкладка `Платежи` позволяет:

- добавить оплату;
- добавить возврат;
- отредактировать платеж;
- распровести платеж.

## Акты

В карточке договора вкладка `Акты` позволяет:

- создать акт;
- открыть акт;
- удалить акт;
- распечатать акт;
- распечатать талоны;
- распечатать акт и талоны вместе.
