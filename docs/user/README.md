# Пользовательская документация

Этот раздел описывает ежедневную работу оператора в RD4.

## Основные разделы приложения

- `Договоры` — реестр договоров, поиск, фильтры, создание, редактирование, печать.
- `Платежи` — оплаты, возвраты и распроведение платежей в карточке договора.
- `Акты` — акты выполненных услуг, печать актов и талонов.
- `Справочник услуг` — дерево папок и медицинских услуг.

В главном меню доступны разделы `Файл`, `Настройки`, `Отчеты` и `Помощь`. В `Помощь` пункт `О программе` показывает краткую информацию о приложении и текущей базе данных.

## Вход в систему

Окно входа содержит:

- выбор пользователя;
- поле пароля;
- кнопку `Войти`;
- служебный блок с папкой данных и путем к базе.

При первом запуске пустой базы создается пользователь:

```text
login: admin
password: admin
```

## Договоры

На главной странице доступны:

- создание договора;
- создание договора на основе выбранного;
- открытие карточки договора;
- редактирование;
- печать договора из контекстного меню;
- soft-delete договора;
- фильтры по периоду, типу оплаты, балансу и видимости.

В списке договоров колонка `Выписка` показывает отметку `Да`, если пациентка выписана.
Если выписанный договор находится в долге или переплате, вся строка подсвечивается светлым оттенком соответствующего статуса.
При открытии программы список договоров прокручивается к последней видимой записи, и эта запись выделяется.

Удаленные договоры скрываются из активного списка, но доступны через фильтр видимости.

В форме создания и редактирования договора телефоны пациентки и представителя вводятся по маске `+7 (000) 000-00-00`.
Обязательные поля подсвечиваются при сохранении, если они не заполнены. Если включен представитель, его данные также обязательны. Нужно выбрать хотя бы один вид услуги: `Стационарное лечение` или `Роды`. Для договора ФОМС обязателен номер ФОМС, для платного договора он может быть пустым.
Клавиша `Tab` последовательно переводит фокус между элементами формы, включая многострочные поля.

## Платежи

В карточке договора над вкладками показывается сводка: начислено, оплачено, возвраты, баланс и статус. Вкладки `Платежи` и `Акты` показывают количество записей.

В карточке договора вкладка `Платежи` позволяет:

- добавить оплату;
- добавить возврат;
- отредактировать платеж;
- распровести платеж.

Над таблицей платежей отображаются итоги по оплатам, возвратам, общей сумме проведённых платежей и числу распроведённых платежей.

## Акты

В карточке договора вкладка `Акты` позволяет:

- создать акт;
- открыть акт;
- удалить акт;
- распечатать акт;
- распечатать талоны;
- распечатать акт и талоны вместе.

Над таблицей актов отображаются количество актов, количество строк услуг и сумма по актам.
